== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin to your blog/website.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your Wildfireapp-SponsorID in the 'Settings' menu in WordPress.

Please see our [tutorial](http://wildfireapp.zendesk.com/entries/384410-widget-setup-for-wordpress-blogs "Installing WordPress Plugin").
